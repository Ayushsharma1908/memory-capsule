# Memory Capsule

A Chrome Extension that captures ChatGPT conversations and exports them as structured JSON files.

## Features

- Detect ChatGPT messages
- Store conversations locally
- Export conversations as JSON
- Foundation for cross-chatbot memory transfer

## Tech Stack

- JavaScript
- Chrome Extension Manifest V3
- Chrome Storage API

## Installation

1. Clone the repository
2. Open Chrome
3. Go to chrome://extensions
4. Enable Developer Mode
5. Click "Load unpacked"
6. Select the project folder
